# Superstore — Interactive Sales Dashboard

Interactive web dashboard built with Streamlit to explore Superstore sales data visually.

## Overview

A data visualization app that lets users explore sales trends, regional performance, and product categories through interactive charts and filters.

## Tech Stack

**Python** · **Streamlit** · **Pandas** · **Plotly / Matplotlib**

## Project Structure

```
├── dashboard.py   # Main Streamlit app
└── a.csv          # Superstore dataset
```

## How to Run

```bash
pip install streamlit pandas plotly
streamlit run dashboard.py
```

Then open your browser at `http://localhost:8501`

## Features

- Sales overview with KPIs
- Regional breakdown
- Category and sub-category analysis
- Time series trends
- Interactive filters by segment and region
